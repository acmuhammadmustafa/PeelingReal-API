Title: Peeling Real

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Overview: Peeling Real is an API project that allows you to login, view items through filtering between categories, or colors, and add items into your shopping cart. If you're an admin, you are able to add, update, or delete categories.
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

How to Run:

1. Open your IDE (IntelliJ or Visual Studio Code, etc.).

2. Clone/Download the project files.

3. Run the "EasyshopApplication.java" file.

4. Go to PeelingReal-Web and open the index-html.

5. Go to the top right corner and click on your preferred browser. 

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Features:

1.) Able to view and filter through items based on category.

2.) Able to view and filter through items based on price.

3.) Able to view and filter through items based on colors.

4.) Able to login.

5.) (Admin required) Add categories.

6.) (Admin required) Remove categories.

7.) (Admin required) Update categories.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
